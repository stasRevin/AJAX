This plugin puts HTML table data into an excel spreadsheet.

To use it, click on the "Export button". When prompted about the file, select "Open" and choose MS Excel as the opening application.

If the browser does not prompt you to select the download method, the file will just be downloaded to your "Dowlnloads" folder.

The file will not have an Excel extension, therefore you will need to explicitly open it with MS Excel.
